Direction Comparison
Dimension A Restrained Utility B Warm and Confident C High Contrast Execution
Visual warmth Low — neutral surfaces, orange as accent only High — amber-cream in headers and quick-add Low in Planning, pronounced shift in Shopping
Mode distinction Subtle — surface elevation change only Moderate — warmer Shopping surface Strong — bolder weights, round checkboxes, orange Done chip
Brand alignment Quiet — orange present but not leading Prominent — orange family throughout Functional — orange reserved for action and completion
Calm confidence feel Strongest — minimal visual noise Good — warmth reads as welcoming, not anxious Good in Planning; Shopping mode is energetic by design
Supermarket legibility Good — clean, adequate size Good — same layout as A Strongest — bolder weights, larger items, maximum fade contrast
Complexity to implement Lowest — stock M3 with minimal overrides Low — container color overrides for key components Moderate — custom weights and checkbox shape in Shopping mode
Best for Users who value simplicity; closest to Mealie web aesthetic Users who want the app to feel warm and kitchen-native Users who prioritize Shopping mode legibility above all

These mockups are HTML approximations of Android UI — proportions and spacing are representative, not pixel-perfect. Actual Jetpack Compose implementation will use Material 3 tokens throughout. All three directions use the same seed color (#E58325), the same type scale (Roboto), and the same adaptive density rules. The choice is about emphasis and personality, not about core functionality.
